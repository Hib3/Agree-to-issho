# Aguri Asset Review Checklist

## 必須確認

- [ ] オリジナル人間少女キャラクター「アグリちゃん」として見える。
- [ ] 猫、動物、マスコット、既存IPキャラクターに見えない。
- [ ] レインボーカラーのボブヘア、青い目、三角歯、小さな紫の帽子、花飾り、リボン、紫パーカー、黒スカート、紫黒ニーソ、黒ブーツが保たれている。
- [ ] 全身が1体だけ入っている。
- [ ] 同一画角、同一立ち位置、同一余白である。
- [ ] 背景は透明である。
- [ ] 白背景の除去漏れ、ドット欠け、白フチが目立たない。
- [ ] 歯、目、リボンなどの白い内部パーツが消えていない。
- [ ] 余計な手足、靴、顔、別ポーズ断片がない。
- [ ] `pending/` で確認してから `approved/` へ移動している。

## 不採用

- [ ] 背景つき、白背景つき、低解像度、切れ、分裂、浮遊パーツがある。
- [ ] 顔や服が正本から大きく変わっている。
- [ ] 既存IPのキャラクター、UI、ロゴ、台詞、設定を連想させる要素がある。
- [ ] 1枚に複数キャラクターや複数ポーズが入っている。
